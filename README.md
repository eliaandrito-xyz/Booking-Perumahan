# Booking-Perumahan
Sistem ini menyediakan fitur manajemen proyek perumahan, termasuk pengelolaan data developer, proyek perumahan, unit rumah, gambar unit, dan fasilitas proyek. Setiap rumah memiliki informasi lengkap seperti nomor unit, tipe rumah, luas tanah, luas bangunan, jumlah kamar, harga, status ketersediaan, dan dokumentasi visual seperti foto dan siteplan
